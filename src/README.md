# Currency Converter App

A React + TypeScript application that converts currencies using live exchange rates.  
Built with **Ant Design** for UI and includes features like searchable dropdowns with flags, input validation, and a countdown timer.

---

## Features

- Fetches live exchange rates from ExchangeRate API.
- Fetches currency names from Open Exchange Rates.
- Searchable dropdown with country flags.
- Input validation for numeric values.
- Convert button to calculate and display result.
- **Countdown Timer**: Result disappears after 10 minutes.
- Responsive UI using **Ant Design**.
- Unit tests with **Jest + React Testing Library**.

---

## Tech Stack

- **React** (with Hooks)
- **TypeScript**
- **Ant Design**
- **Jest + React Testing Library** for unit tests
